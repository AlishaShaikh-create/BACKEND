db.student.find({city:"Delhi",marks:75});

-> mongo db have may operators 
$eq -> matches values that are eqaul to a specific value

$gt -> greater than 

$gte-> greater than or equal to

EXAMPLE:

-> db.student.find({marks:{ $gt:75 } } );

-> db.student.find({city:{$in : [ 'delhi':'mumbai']}})

-> mongodb value are case sensative 

student who scored > 75 or live in Delhi

db.student.find( { $or : [ { marks:{$gt:75}},{city:"Delhi"} ] } )

UPDATE IN DATABASE

db.collection.updateOne() --> Update at most a single document that matches a specified filter even though multiple document may matches the  specified filter.

db.collection.updateOne( <filter> , <update>,<option>)

db.student.updateOne( { name:"Adam"},{$set : { marks : 99 }})

db.collection.updateMany()
